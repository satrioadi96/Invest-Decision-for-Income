# Invest-Decision-for-Income
the simple program in Bahasa (ID/MY) for choose decision amount investation from the income


# Included Variable (ID/MY)
1.  Pendapatan Upah/Gaji (Personal Income)
2.  Kebutuhan (Needed Item)
3.  Waktu berkala : Harian/Bulanan (Periodic Time : Days/Months)
4.  Waktu Kerja (Weekday)
5.  Tabungan/Investasi (Investation)
6.  Keputusan (Decision)
